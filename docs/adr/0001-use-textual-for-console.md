# 0001: Use Textual for Console

We have decided to use the [Textual](https://textual.textualize.io/) framework to build the **Console**. This choice supports our goal of an "out of this world" and "elegant" UI by providing sophisticated layout management, CSS-based styling, and a robust event-driven architecture, which would be significantly more difficult to achieve with simpler libraries like `Rich` or raw terminal manipulation.
